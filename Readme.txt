Hook useEffect yang dimana memiliki kekurangan jika pada saat development menggunakan React.Strictmode, karena
jika menggunakan useEffect default nya akan mengeluarkan output mounting sebanyak 2 kali.

Oleh karena itu bisa menggunakan React Query untuk menggantikanya. useQuery juga dapat untuk melakukan fetch data.
dan kelebihan dari penggunaan react query yaitu ketika kita menutup/beralih ke jendela web lain dan ketika kita akan,
kembali kepada halaman web kita makan dengan menggunakan react query akan melakukan fetch data yang terbaru.
